## Le papillon

Rien n’arrête un papillon !

### Tutoriel

Tu peux déplacer le papillon :

```
m 2.5,2.5
```

Dessiner des lignes verticales :

```
v2.5
```

Horizontales :

```
h2.5
```

Des lignes :

```
l 2.5,2.1
```

Des arcs de cercle :

```
a2.5,2.5 0 0,1 2,-1.5
```

Et enfin, fermer le chemin tracé :

```
Z
```

Envole-toi papillon !

```
m5,5
a 1,1 300 0,1 -1,-1
a 1,1 300 0,1 -1,-1
m2,2
a 1,1 60 1,1 -1,-1
a 1,1 60 1,1 -1,-1
m3,3l7,7
```
